Fast R-CNN目标检测方法介绍

发表于 2018-07-22 | 分类于 深度学习

提出的背景

顾名思义，Fast R-CNN就是为了克服R-CNN训练慢的问题，R-CNN主要体现在一下三个方面的问题：

  • 整个模型分为多个步骤，包括Selective Search提取的目标候选区，训练CNN特征提取模型，训练SVM分类器，训练边框回归器
  • 测试时间长，由于每张图片需要处理大量的目标候选框
  • 训练时所需空间大，花费的时间多，R-CNN在训练时每个候选区域都要调整成相同大小的图像，并输入到CNN的网络中，这使得处理一张图片所需的空间大，整个模型需要训练CNN模型，SVM分类器以及目标边界回归器，所以训练时间花费很多。

优缺点

优点：

  • 节省时间，主要表现在Fast R-CNN将整幅图像输入到网络中去，在网络正向传播训练的过程中提取候选区域
  • 不需要过大的磁盘进行存储，主要表现在不需要将候选区的特征向量保存起来，将类别判断和位置精调学习统一起来，并不是像R-CNN是多个阶段的训练

缺点：

  • 仍然用到了候选区域生成的算法，需要识别的区域很多
  • 无法融入GPU，在一定程度上影响了整个算法的效率
